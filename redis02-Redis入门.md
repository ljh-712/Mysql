# Redis入门

## 概述

> Redis是什么

Remote Dictionary Server,远程字典服务

免费开源，C语言编写

支持网络，可就内存亦可持久化的日志型

> Redis能干嘛

1. 内存存储，持久化
2. 效率高，高速缓存
3. 地图信息分析
4. 发布订阅系统
5. 计时器，计数器（浏览量）

> 特性

1. 多样的数据类型
2. 持久化
3. 集群
4. 事务
